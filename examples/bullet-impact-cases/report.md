# Abaqus Explicit Dynamics 仿真报告

## 实验概述

使用 Abaqus/Explicit 进行两种弹丸冲击仿真验证：
- **示例一**：7.62mm 弹丸击穿 10mm 厚钢板
- **示例二**：7.62mm 弹丸径向击穿管道（撞击点在轴线方向中心）

---

## 示例一：子弹击穿钢板

### 模型参数

| 参数 | 值 |
|---|---|
| 弹丸直径 | 7.62 mm |
| 弹丸长度 | 30 mm（平头圆柱等效） |
| 弹丸材料 | 超高强度钢（E=210 GPa, ν=0.3, ρ=8000 kg/m³） |
| 弹丸速度 | 850 m/s（步枪弹典型速度） |
| 钢板尺寸 | 200 × 200 × 10 mm |
| 钢板材料 | 结构钢（E=210 GPa, ν=0.3, ρ=7850 kg/m³） |
| 塑性模型 | 双线性等向强化（屈服强度 400 MPa） |
| 时间周期 | 2.0e-4 s（200 μs） |
| 单元类型 | C3D8R（8节点六面体减缩积分） |

### 网格信息

| 指标 | 值 |
|---|---|
| 单元总数 | 50,270 |
| 节点总数 | 61,606 |
| 自由度（DOF） | 184,818 |

### 计算结果

| 指标 | 值 |
|---|---|
| 最大 Mises 应力 | **2.13 GPa** |
| 输出帧数 | 51 |
| 分析耗时 | ~106 秒 |
| 最终仿真时间 | 2.0000E-04 s（完整模拟） |

### 结果分析

弹丸以 850 m/s 高速撞击钢板，接触区域产生显著的应力集中，最大等效应力达 2.13 GPa，远超材料的屈服强度（400 MPa），表明钢板发生了显著的塑性变形和可能的破坏。

---

## 示例二：子弹径向击穿管道

### 模型参数

| 参数 | 值 |
|---|---|
| 弹丸直径 | 7.62 mm |
| 弹丸长度 | 30 mm |
| 弹丸速度 | 850 m/s |
| 管道外径 | 100 mm |
| 管道壁厚 | 10 mm（内径 80 mm） |
| 管道长度 | 300 mm |
| 管道材料 | 结构钢（E=210 GPa, ν=0.3, ρ=7850 kg/m³） |
| 撞击位置 | 径向，轴线方向中心（Z=150 mm） |
| 时间周期 | 3.0e-4 s（300 μs） |

### 网格信息

| 指标 | 值 |
|---|---|
| 单元总数 | 39,370 |
| 节点总数 | 49,486 |
| 自由度（DOF） | 148,458 |

### 计算结果

| 指标 | 值 |
|---|---|
| 最大 Mises 应力 | **2.11 GPa** |
| 输出帧数 | 51 |
| 分析耗时 | ~143 秒 |
| 最终仿真时间 | 3.0000E-04 s（完整模拟） |

### 结果分析

弹丸径向撞击管道壁面，最大 Mises 应力达 2.11 GPa。应力水平与钢板案例接近（2.13 GPa），表明两种工况下的冲击载荷相似。管道在撞击点局部产生严重塑性变形。

---

## 对比总结

| 指标 | 钢板 | 管道 |
|---|---|---|
| 网格规模（单元） | 50,270 | 39,370 |
| 网格规模（节点） | 61,606 | 49,486 |
| 最大 Mises 应力 | 2.13 GPa | 2.11 GPa |
| 仿真时间周期 | 200 μs | 300 μs |
| 分析耗时 | ~106 s | ~143 s |

两种仿真均成功完成，弹丸有效穿透目标结构，Mises 应力水平合理一致。
